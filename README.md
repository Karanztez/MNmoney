


### 💰 **MNmoney: Magnus Network Economy System**

**MNmoney** เป็นปลั๊กอินระบบเศรษฐกิจ (Economy) ประสิทธิภาพสูงสำหรับเซิร์ฟเวอร์ Minecraft (Paper/Spigot) ที่ถูกออกแบบมาเพื่อเน้นความรวดเร็ว มั่นคง และรองรับการขยายตัวของเซิร์ฟเวอร์ (Scalability) ในระยะยาว



### ✨ **Features (คุณสมบัติเด่น)**

* 🚀 **High Performance:** ขับเคลื่อนด้วยระบบ Write-behind Caching ทำให้การตรวจสอบยอดเงินรวดเร็วขั้นสุด การันตีเซิร์ฟเวอร์ลื่นไหล ไม่กระตุก (No Lag) แม้จะมีผู้เล่นออนไลน์พร้อมกันจำนวนมาก
* 💾 **Hybrid Storage:** ยืดหยุ่นด้วยทางเลือกการจัดเก็บข้อมูล สามารถเลือกใช้ **MySQL** (เหมาะสำหรับระบบ Network/BungeeCord) หรือ **Local JSON** (เหมาะสำหรับเซิร์ฟเวอร์ทั่วไป)
* 🏦 **Wallet & Bank System:** จัดการการเงินได้คล่องตัวขึ้น ด้วยระบบเงินติดตัว (Wallet) และระบบฝากเงินในธนาคาร (Bank)
* 🔗 **Vault Integration:** เชื่อมต่อและทำงานร่วมกับปลั๊กอินยอดฮิตอย่าง ShopGUI+, ChestShop และอื่นๆ ผ่าน Vault API ได้อย่างสมบูรณ์แบบ
* ⚔️ **Mob Money Drops:** เพิ่มความสนุกให้ผู้เล่นด้วยระบบมอนสเตอร์ดรอปเงินเป็นไอเทมเหรียญทอง (สามารถปรับแต่ง Custom ได้ตามต้องการ)
* 📊 **Leaderboard:** ระบบจัดอันดับเศรษฐีผู้เล่นที่รวยที่สุด (Top 10) อัปเดตข้อมูลแบบ Real-time
* 📜 **Transaction Logs:** บันทึกทุกความเคลื่อนไหวทางการเงิน ป้องกันปัญหาและสามารถตรวจสอบย้อนหลังได้อย่างโปร่งใส *(ใช้งานได้เฉพาะโหมด MySQL)*

### 🎮 **Commands & Permissions (คำสั่งและสิทธิ์การใช้งาน)**

#### 👤 **Player Commands (คำสั่งสำหรับผู้เล่นทั่วไป)**

| คำสั่ง (Command) | 📝 รายละเอียด (Description) | 🔑 สิทธิ์ (Permission) |
| --- | --- | --- |
| `/mnc` | ดูยอดเงินในกระเป๋า (Wallet) และธนาคาร (Bank) | - |
| `/mnp <player> <amount>` | โอนเงินให้ผู้เล่นอื่น | - |
| `/mndeposit <amount>` | ฝากเงินเข้าธนาคาร | - |
| `/mnwithdraw <amount>` | ถอนเงินจากธนาคาร | - |
| `/mntop` | ดูอันดับผู้เล่นที่รวยที่สุด | `mnmoney.top` |

#### 🛠️ **Admin Commands (คำสั่งสำหรับผู้ดูแลระบบ)**

| คำสั่ง (Command) | 📝 รายละเอียด (Description) | 🔑 สิทธิ์ (Permission) |
| --- | --- | --- |
| `/givemoney <player> <amount>` | เพิ่มเงินให้ผู้เล่น | `mnmoney.admin` |
| `/setmoney <player> <amount>` | ตั้งค่ายอดเงินผู้เล่น | `mnmoney.admin` |
| `/mnadmin export` | ย้ายข้อมูลระบบจาก Local JSON เข้าสู่ MySQL | `mnmoney.admin` |
